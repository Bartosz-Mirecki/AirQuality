# AirQuality

# General info
This project shows prediction of the air quality parameter(tin oxide) 24 hours ahead .
	
# Technologies
Project is created with:
* tensor flow version: 2.9.1
* numpy version: 1.23.4
* streamlit version: 1.13.0
* matplotlib version: 3.3.4
